## 🎮 游戏流程

1. **启动应用** → 显示主菜单
2. **选择今日挑战** → 加载每日谜题
3. **游戏操作**：
   - 拖动卡片到合适位置
   - 旋转卡片匹配文字
   - 通过镂空发现单词
4. **完成条件**：发现所有单词组成完整句子
5. **奖励**：记录完成时间，更新统计

## 🔧 配置选项

```typescript
{
  // 游戏设置
  TIMER_DURATION: 180,        // 时间限制（秒）
  GRID_SIZE: 12,              // 网格大小
  CARD_WIDTH: 6,              // 卡片宽度
  CARD_HEIGHT: 4,             // 卡片高度
  
  // 音效设置
  soundEnabled: true,         // 音效开关
  hapticEnabled: true,        // 震动开关
  juicyEffectsEnabled: true,  // 特效开关
  
  // 难度设置
  difficulty: 'medium',       // easy/medium/hard
}
```
