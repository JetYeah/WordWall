## 游戏概述
一款每日文字解谜游戏，玩家通过带有镂空的解密卡从文字墙上框出有意义的信息，解开每日谜题。

## 核心玩法
1. **每日挑战**：每天一道新题，谜底为温暖积极的话语
2. **卡片机制**：玩家获得独特镂空卡片，可横向、纵向放置或旋转
3. **文字页面**：包含谜底和干扰信息的文字网格
4. **时间限制**：3分钟内完成，超时失败
5. **历史挑战**：可挑战历史题目，成功后记录完成时间

## 技术栈建议
- **前端**：React Native (iOS/Android) + React (Web)
- **后端**：Node.js + Express
- **数据库**：Firebase Firestore (存储题目、用户进度)
- **音效**：Howler.js
- **动效**：React Native Reanimated / Framer Motion
