字垣 · 品牌资产生成器

指纹算法逐字移植自 src/game/fingerprint.ts；调色板取自 BookmarkCard.tsx / config.ts。

用途：本页用 canvas 按精确尺寸渲染全部品牌资产，点「下载」保存 PNG 到 assets/。
图标族（icon / adaptive 前景·背景·单色 / favicon）已由 node scripts/gen-brand-assets.js 直接生成；本页可调整种子/角度/用时后重新下载换一种观感。
启动页 splash.png 含中文（字垣 + 标语）必须在此页用浏览器字体渲染后下载。
生效：改完 app.json 与 assets/ 后，需 expo prebuild / EAS / run:android 重新构建；Expo Go 不会显示自定义图标/启动页。
